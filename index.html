演示流程

步骤 1：写入 NFC 链接
板端通过 nfc_box_url 0 生成并写入带柜号参数的链接。
步骤 2：手机贴近板子
手机自动识别 NFC 标签，并打开对应柜体网页入口。
步骤 3：网页解析参数
页面根据 URL 中的 box 与 mode 参数显示目标柜体信息。
步骤 4：模拟开柜
点击按钮展示前后端联动的目标逻辑，后续可扩展为真实接口调用。

比赛展示建议

  • 把 box=0、box=1 等参数和真实柜号对应起来。
  • 后续可增加取件码、人脸识别结果、柜门状态同步。
  • 部署到公网后，把 NFC 写入地址替换成真实网页域名。
